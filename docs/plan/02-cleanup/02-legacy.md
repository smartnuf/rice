# 02-cleanup / 02 — Remove legacy implementation

Status: `todo`

## Goal

Remove old implementation paths once the current implementation is validated.

## Tasks

- Delete legacy source files.
- Delete or migrate legacy-only tests.
- Delete or update legacy documentation.
- Update imports and examples.
- Run the full test suite.

## Done means

- No dead legacy implementation remains.
- Tests confirm the supported implementation still works.
- Documentation no longer points users to removed code.
